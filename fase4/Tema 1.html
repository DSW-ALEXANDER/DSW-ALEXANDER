Frameworks CSS

¿Qué es?

Son herramientas y pautas para desarrollar al estilo de un sitio web, suele usarse como un uso comun para sustituir elementos predeterminados del navegador, tambien ayuda a hacer el texto accesible en cualquier soporte, como tambien crea cualquier layout robusto facilmente, También son hojas de de estilo de cascadas y se refiere a una técnica utilizada para determinar la apariencia de una pagina web, tenemos por ejemplo el HTML que vienen siendo las paredes de la casa, y el código es Css, que vendría siendo la pintura y las cortinas que hacen que el HTML adquiera una apariencia atractiva, a simple vista la plantilla lo incluye todo, desde el color de los encabezados o los enlaces, hasta la posición de los elementos dentro de la página web.

¿Porque usar frameworks CSS?

Al usar esta herramienta le da por decirlo así, otro sentir, otra visualización mucho mejor, que en palabras técnicas se puede decir que para lograrlo se debe utilizar una hoja de estilo con Css, l dicho menú adquiere un aspecto más elegante, en conclusión, el poder de usar framework radica en que permite darle forma, estilos y permite cambiar la plantilla del sitio web sin tener que variar el código html.
